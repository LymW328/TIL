### 파이썬에서 '복사'
-   할당 
    -   대입 연산자(=)

        같은 주소를 참조하기 때문에 오류가 발생할 수 있다.
        
        이를 해결하기 위해 ':' 사용(단, 1차원 리스트에서만 유효!)
        왜냐하면 리스트 안의 리스트도 고유의 주소를 가지고 있기 때문!
```
a = [1,2,3]
b = a[:] #새로운 주소에 할당! 슬라이싱 안하면 b를 수정할 때 a 도 변경된다.
print(a,b)
b[0] = 5
print(a,b)

```
-   2차원적인 리스트에서 얕은 복사 문제 해결
    -   b = copy.deepcopy(a) 사용
```
import
```
