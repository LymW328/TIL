def pivo(n):
    if n == 1:
        return 1
    elif n == 2:
        return 1
    else:
        return pivo(n-1)+pivo(n-2)

```python
#하노이의 탑? 아직 이해 불가능...
def hanoi(n, start, end, other): #원판의 갯수, 시작, 도착, 서브 기둥 번호?
    if n ==1:
        print(f"{start} -> {end}")
        return
    else:
        hanoi(n-1, start, end, other)
        print(f'{start} -> {end}')
        hanoi(n-1, other, end, start)
```

함수에 print의 f-string 처럼 입력란을 쓰는 방법을 모르겠다.
